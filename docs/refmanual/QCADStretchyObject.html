QCADStretchyObject

QCADStretchyObject — Base class for all resizable design objects

Synopsis




            QCADStretchyObject;

Object Hierarchy


  GObject
   +----QCADObject
         +----QCADDesignObject
               +----QCADStretchyObject
                     +----QCADLabel
                     +----QCADRuler
                     +----QCADSubstrate

Description

QCADStretchyObject provides default mouse handlers for creating resizable objects. Once such an object has been created, these mouse handlers are responsible for selecting it and resizing it. This latter functionality has not yet been implemented.
