Description

QCADDesignObject is the common base class for all design objects. It defines the necessary virtual functions for serialization, bounds calculation, selection, drawing, and transformation.

qcad_design_object_class_get_PostScript_preamble ()

const char* qcad_design_object_class_get_PostScript_preamble
                                            (QCADDesignObjectClass *klass);

This function returns a string containing the definition of a PostScript function which renders a QCADDesignObject of this type into a PostScript stream. This function may make use of the PostScript functions defined in PS_TEXT_PLACEMENT_PREAMBLE, as well as PrintMagic() and the per-page preamble.

See also: print_world().

klass : QCADDesignObjectClass whose PostScript preamble to retrieve
Returns : the PostScript preamble for this class' GType as a string.

qcad_design_object_get_PostScript_instance ()

char*       qcad_design_object_get_PostScript_instance
                                            (QCADDesignObject *obj,
                                             gboolean bColour);

This function returns a string containing a PostScript function call to the function returned by qcad_design_object_get_PostScript_preamble() for this object type. The parameters to the function may be defined in terms of calls to the PostScript functions defined in PS_TEXT_PLACEMENT_PREAMBLE, as well as PrintMagic() and the per-page preamble.

See also: print_world().

obj : QCADDesignObject to render as PostScript.
bColour : Whether to render it in colour.
Returns : the PostScript instance function call for this object as a string.
