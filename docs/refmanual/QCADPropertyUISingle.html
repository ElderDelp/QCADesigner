Description

A "single" propert UI covers only one of a GObject instance's properties, identified by its name. It creates a 2D grid of widgets that allow you to set the instance's property to any value allowed by its GParamSpec.

In addition, if the instance happens to be of type QCADObject, the property and behaviour hints pertaining to the instance property being covered are retrieved from the instance's class and applied.

QCADPropertyUISingle

typedef struct _QCADPropertyUISingle QCADPropertyUISingle;

qcad_property_ui_single_newv ()

QCADPropertyUI* qcad_property_ui_single_newv
                                            (GObject *instance,
                                             char *property,
                                             va_list va);

Creates a new property UI exposing the property property of GObject instance instance.

If instance happens to be of type QCADObject, then this function looks up those QCADPropertyUIBehaviour entries which connect the new property UI's properties to instance's properties and makes the appropriate connections.

If instance happens to be of type QCADObject, then this function looks up those QCADPropertyUIProperty entries which apply to the new property UI's properties and sets the values as prescribed in the matching QCADPropertyUIProperty entries.

See also: qcad_property_ui_new(), QCADPropertyUIBehaviour, QCADPropertyUIProperty

instance : Instance to create the UI for.
property : instance's property to create the UI for.
va : va_list of NULL-terminated property name - property value pairs to set on the newly created UI.
Returns : A newly created QCADPropertyUISingle for property property of GObject instance instance.
