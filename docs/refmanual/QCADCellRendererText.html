Description

GtkCellRendererText widgets in versions of GTK <= 2.4.0 had neither a "sensitive" property, nor an "editing-started" signal. The signal documented here as "workaround-editing-started" will be defined in your application as "editing-started", and the "sensitive" property will also be defined if, at compile time, the version of GTK is found to be <= 2.4.0.

For versions of GTK > 2.4.0 detected at compile time, the definition of this widget evaluates to GtkCellRendererText and none of the QCADCellRendererText code gets compiled.
