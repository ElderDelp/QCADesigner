Creating and Importing Blocks

The ability to manipulate blocks of cells allows you to import other designs or chunks of other designs into the current one, as well as create blocks from chunks of the current design.

Create Block

To create a block, select all the cells in the design you wish to use in the block. TIP: You can use SHIFT to add to the selection, as well as CTRL to remove from the selection. Once you have selected all the cells, click Create Block from the Tools menu. This will allow you to choose a filename for the block consisting of all your selected cells.

Import Block

Allows you to select any QCADesigner design file. This design will be imported into the current design and will become a new selection you can drag to a place of your choosing.
