Property Details

The "amplitude" property

  "amplitude"            gdouble               : Read / Write

Clock amplitude.

Default value: 1

The "dc-offset" property

  "dc-offset"            gdouble               : Read / Write

Clock DC offset.

Default value: 0

The "frequency" property

  "frequency"            gdouble               : Read / Write

Clock frequency.

Allowed values: >= G_MINDOUBLE

Default value: 1e+06

The "max-clock" property

  "max-clock"            gdouble               : Read / Write

Maximum clock voltage.

Default value: 0

The "min-clock" property

  "min-clock"            gdouble               : Read / Write

Minimum clock voltage.

Default value: 0

The "phase" property

  "phase"                gdouble               : Read / Write

Clock phase.

Allowed values: [0,360]

Default value: 270

The "relative-permittivity" property

  "relative-permittivity" gdouble               : Read / Write

Relative permittivity of the environment between the electrode and the ground.

Allowed values: >= 1

Default value: 1

The "thickness" property

  "thickness"            gdouble               : Read / Write

Electrode thickness in z direction.

Allowed values: >= 1

Default value: 1

The "z-to-ground" property

  "z-to-ground"          gdouble               : Read / Write

Distance from electrode to ground electrode.

Allowed values: >= 1

Default value: 1
