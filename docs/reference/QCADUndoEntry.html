Description

This object incorporates the basic functionality necessary for undo/redo.

You create a QCADUndoEntry with qcad_undo_entry_new(), connect to its "apply" signal, and push the QCADUndoEntry into a QCADUndoEntryGroup using qcad_undo_entry_group_push_group().

qcad_undo_entry_fire ()

void        qcad_undo_entry_fire            (QCADUndoEntry *undo_entry,
                                             gboolean bUndo);

Causes the QCADUndoEntry to "fire". In such a case, a QCADUndoEntry will emit the "apply" signal, calling any attached handlers.

undo_entry : Entry to fire
bUndo : Whether to undo (TRUE) or redo (FALSE)

Signal Details

The "apply" signal

void        user_function                  (QCADUndoEntry *entry,
                                            gboolean       bUndo,
                                            gpointer       user_data)      : Run first

This signal is emitted whenever entry is activated by a QCADUndoEntryGroup.

entry : Undo entry
bUndo : Whether to undo (TRUE) or redo (FALSE)
user_data : user data set when the signal handler was connected.
