Properties


  "render-as"            guint                 : Read / Write

Description

Creates a property UI for integer properties. It can be rendered as a spin button or a combo box.

Note: Rendering the property UI as a combo for a property having a large range of values is a bad idea.

Property Details

The "render-as" property

  "render-as"            guint                 : Read / Write

This property UI can be rendered either as a GtkSpinButton or as a GtkComboBox. Note that, if you render it as a GtkComboBox and the range of the instance's GParamSpec is too large, the resulting GtkComboBox popup menu will be enormous and will likely take a long time to construct. When in doubt, leave "render-as" set to its default: GtkSpinButton.

Valid values: GTK_TYPE_SPIN_BUTTON, GTK_TYPE_COMBO_BOX
