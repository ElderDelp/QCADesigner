QCADPropertyUIDouble

QCADPropertyUIDouble — Property UI for exposing single and double precision floating point properties

Synopsis




            QCADPropertyUIDouble;

Object Hierarchy


  GObject
   +----QCADPropertyUI
         +----QCADPropertyUISingle
               +----QCADPropertyUINumeric
                     +----QCADPropertyUIDouble

Properties


  "digits"               guint                 : Read / Write
  "step-increment"       gdouble               : Read / Write

Description

Creates a property UI for properties with a single or double precision floating point value type.

Details

QCADPropertyUIDouble

typedef struct _QCADPropertyUIDouble QCADPropertyUIDouble;

Property Details

The "digits" property

  "digits"               guint                 : Read / Write

Number of decimal digits to display.

Allowed values: <= 10

Default value: 2

The "step-increment" property

  "step-increment"       gdouble               : Read / Write

Amount to modify value by.

Allowed values: >= 0

Default value: 0.01
