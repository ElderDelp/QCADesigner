Property Details

The "render-as" property

  "render-as"            guint                 : Read / Write

This property UI can be rendered as either a GtkCheckButton (default) or a GtkToolItem (a GtkToggleToolButton).

Valid values: GTK_TYPE_CHECK_BUTTON, GTK_TYPE_TOOL_ITEM

The "stock-down" property

  "stock-down"           gchararray            : Read / Write

  "stock-up"             gchararray            : Read / Write
