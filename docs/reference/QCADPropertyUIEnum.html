The "render-as" property

  "render-as"            guint                 : Read / Write

This widget can be rendered as either a GtkComboBox (default), or as a set of GtkRadioButton widgets inside a GtkFrame.

Valid values: GTK_TYPE_COMBO_BOX, GTK_TYPE_RADIO_BUTTON
