Property Details

The "show-potential" property

  "show-potential"       gboolean              : Read / Write

Show potential created by the electrodes on this layer.

Default value: FALSE

The "tile-size" property

  "tile-size"            guint                 : Read / Write

Resolution (n x n pixels) used to draw the potential.

Allowed values: >= 1

Default value: 16

The "time-coord" property

  "time-coord"           gdouble               : Read / Write

Time coordinate to draw the potential for.

Allowed values: >= 0

Default value: 0

The "z-showing" property

  "z-showing"            gdouble               : Read / Write

Distance from clocking layer to show the cross-section for.

Default value: 1
